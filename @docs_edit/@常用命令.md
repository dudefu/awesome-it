# Vscode常用

**VScode基础命令**

- 打开命令栏 ```ctrl+shift+p```

## Vscode同步到印象笔记

格式要求

```python
# 开头必须为如下
---
title: 文章标题
tags: 标签1 标签2(逗号分隔)
notebook: 笔记本名称
---
```

命令

- 同步当前笔记到印象笔记 ```alt+p```

## markdownlint规范

使用命令

- 查看所有警告 ```Ctrl+Shift+M```
- 单个警告，下方自带 黄色线条
- 查看单个警告  ```F8或Shift+F8```

文件目录结构说明

----------

```
fdm
├── base            基础参数文件
├── database        数据库、ES资源配置
├── model           系统数据结构模型
├── tools           系统通用功能目录
├── tests           系统测试目录
├── tmp             临时文件目录，会随时删除
├── statics         静态文件,不会删除
├── Study           私有git库 - Study
├── git             公有git库
├── module          功能模块
├── private         私人文件
└── using_cmd.py    常用命令

```
